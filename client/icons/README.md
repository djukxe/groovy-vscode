# Extension Icon

This directory should contain a 128x128 PNG icon file named `icon.png` for the VS Code extension.

## Icon Requirements
- **Filename**: `icon.png`
- **Size**: 128x128 pixels
- **Format**: PNG
- **Design**: Should represent Groovy language support, perhaps featuring:
  - A stylized "G" for Groovy
  - Blue color scheme (#4298B8 to #2D5F73 gradient)
  - Clean, professional appearance suitable for VS Code marketplace

## Creating the Icon
You can create this icon using:
- Graphic design software (Photoshop, GIMP, etc.)
- Online icon generators
- The provided SVG template (`icon.svg`) converted to PNG

## Temporary Placeholder
Until the actual icon is created, the extension will use a default icon in VS Code.
